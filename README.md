# Изучаем node-js

- Http сервер с роутингом (http-nodejs)
  
    `node ./http-nodejs`

- Http сервер на express (http-express)

    `cd ./http-express`

    `yarn`

    `yarn start`

- Статические файлы (static-path)

    `cd ./static-path`

    `yarn`

    `yarn start`

- Шаблонизаторы (template-engines)

    `cd ./template-engines`

    `yarn`

    `yarn start`
